## SW특기자용 포트폴리오 구성 예시
**(고3 기준, 완성도 높은 2~3개)**

### 1. 포트폴리오 기본 구조

* **Cover (1페이지)**
    * 이름 / 지원 학과
    * 사용 언어 & 기술 스택
    * 관심 분야 (예: 알고리즘, 데이터, AI, 웹)
    * **한 줄 컨셉 필수**
        * -> "실제 문제 해결을 목표로 프로젝트를 수행했습니다."

---

* **프로젝트 1 (핵심 메인 프로젝트)**
    * **※ 가장 중요**
    * **주제 예시**
        * 학교 데이터 분석 시스템
        * 알고리즘 시각화 프로그램
        * 웹/앱 제작
    * **필수 구성**
        * 문제 정의
        * 설계 구조
        * 사용 기술
        * 핵심 코드 설명
        * 성능 개선 시도
        * 한계 & 개선 방향
    * **✓ 면접 질문 70% 여기서 나옴**

---

* **프로젝트 2 (보조 프로젝트)**
    * 알고리즘 문제 해결 기록
    * 미니 게임
    * 데이터 시각화
    * **역할**
        * -> "전공 흥미 + 지속성 증명"

---

* **알고리즘 학습 기록**
    * 백준 / 프로그래머스
    * 난이도별 문제 수
    * 대표 문제 3개 설명
    * **✓ 풀이 사고과정 설명 필수**

---

* **탐구/보고서**
    * 알고리즘 성능 비교
    * 언어별 실행 속도 실험
    * **✓ 있으면 서류 가산점 큼**

---

* **마무리 페이지**
    * "입학 후 배우고 싶은 것 + 연결"

---

* **✓ 포트폴리오 작성 핵심 요약**
    * 코드보다 **설명**
    * 결과보다 **과정**
    * 성공보다 **개선 시도**
